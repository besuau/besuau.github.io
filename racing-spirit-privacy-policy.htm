RACING SPIRIT PRIVACY POLICY


Racing Spirit does not read or collect any data from your computer but for the Windows user name.

Your Windows user name is only used to init the player name, that you can change later. Your Windows user name is not stored or sended anywhere.

Racing Spirit does not use any internet connection. It does not modify any file in your computer. It just run the game.
